<h1 align="center"> Business Card Generator </h1>
 

 ## A Business Card é um gerador de cartão de visita onde você o acessa por um QrCode.



## :books: Requisitos


- [**Node.js**](https://nodejs.org/en/)
- [**Yarn**](https://yarnpkg.com)

## Iniciando o projeto

```bash

  # Clonar o projeto:

  $ git clone https://github.com/matheusmagnani/businessCardGenerator

  # Entrar no diretório web

  $ cd businessCardGenerator/web
  
   # Instalando as dependências
   
  $ yarn 

  # Iniciar web

  $ yarn dev
  
  # Entrar no diretório server
  
  $ cd businessCardGenerator/server
  
  # Instalar as dependências
   
  $ yarn 
  
  # Iniciar server

  $ yarn dev

```

## 🎨 Protótipo

Você pode colocar suas informações como 'Nome', 'URL do Github', 'URL do LinkedIn', e contar um pouco mais da sua 'História'.

<h2>Irá ser gerado um QrCode que você pode baixar e mostra-lo para quem você preferir.</h2>

Quando scaneado o QrCode ele redireciona para sua página de cartão de visita, contendo todas as informações colocadas já com um estilo incrível e pronto para usar.
